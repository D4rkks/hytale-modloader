## ✨ Planned Features

### 🔑 Account Management & Security
- **Hytale Login:** Secure OAuth2 authentication for official Hytale access.
- **Multi-Account Support:** Quickly switch between different accounts without re-logging.
- **Offline Support:** Able to play with a offline account (but you cant use server functions yet)

### 📦 Instance Management (Prism-Style)
- **Total Isolation:** Each instance maintains its own folders for mods, screenshots, saves, and settings.
- **Mod Downloading:** Native integration with mod repository APIs (Modrinth/CurseForge) for one-click installations.
- **Custom Versions:** Easily manage Alpha, Beta, or Stable Hytale builds.

### 🌐 Social & Multiplayer
- **Discord Rich Presence:** Show your friends exactly what you're doing (current biome, level, or server status).
- **Server Manager:** A dedicated server list with real-time status monitoring (ping, player count) before launching the game.
- **Discord Linking:** Synchronize your launcher profile with your Discord identity.

### 🗺️ World Tools
- **Advanced Creation:** Simplified UI to configure world generation parameters before booting the game.
- **Auto-Backup:** Snapshot system to prevent progress loss in single-player worlds.
